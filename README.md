# FluidWebKit `v10.0`
Fluid WebKit has a set of easy to use helper classes to quickly build custom front-ends using a JSF framework.

The required libraries for the kit are as follows;

    <tr><th>Software</th><th>Version</th></tr>
    <tr><td>JDK</td><td>v11</td></tr>
    <tr><td>JSF (Jakarta Faces)</td><td>v4.0</td></tr>
    <tr><td>PrimeFaces</td><td>v15</td></tr>
    <tr><td>Fluid-API</td><td>v10.0</td></tr>
    <tr><td>Fluid-WS</td><td>v10.0</td></tr>
    <tr><td>CDI (Jakarta CDI)</td><td>v4.0</td></tr>
</table>
